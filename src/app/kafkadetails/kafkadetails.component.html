▪ In large scale cluster migration from Kafka cluster to IBM event stream one of the key task is to migrate all the topics with all relevant configurations. There no out-of-box tools available for performing seamless migration specifically to Event Stream.


▪ Kafka Object Replicator helps in auto replication of Kafka Objects like Topic, Partitions and Replication Factor to IBM Event stream cluster. It provide a Web Interface where user can choose source Kafka Objects and perform replication. This Helps in reducing manual effort and errors. Tool also provides functionality to Customize Object Configuration while replication and generates reports detailing out objects that are migrated successfully or errored
